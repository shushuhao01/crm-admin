# Admin管理后台 - 快速启动指南

## 🚀 快速启动

### 1. 启动后端服务

```bash
# 在项目根目录
cd backend
npm run dev
```

后端服务启动在: http://localhost:3000

### 2. 启动Admin前端服务

```bash
# 在项目根目录
cd admin
npm run dev
```

Admin前端启动在: http://localhost:5174

### 3. 访问管理后台

打开浏览器访问: http://localhost:5174

**登录信息:**
- 用户名: `admin`
- 密码: `admin123`

## 📋 环境要求

- Node.js >= 16
- MySQL >= 5.7
- npm 或 yarn

## 🔧 配置文件

### 开发环境配置

`admin/.env.development`:
```env
VITE_API_BASE_URL=/api/v1/admin
VITE_APP_TITLE=CRM管理后台 - 开发环境
VITE_APP_ENV=development
```

### 生产环境配置

`admin/.env.production`:
```env
VITE_API_BASE_URL=/api/admin
```

## 🛠️ 常见问题

### 1. Network Error / 连接失败

**原因**: 后端服务未启动或端口被占用

**解决**:
```bash
# 检查后端服务是否启动
cd backend
npm run dev

# 检查端口占用
netstat -ano | findstr :3000
```

### 2. 登录失败 - 401错误

**原因**: 用户名或密码错误，或Admin用户未创建

**解决**:
```bash
# 创建Admin用户
node backend/create-admin-user.js
```

### 3. 前端页面空白

**原因**: Admin前端服务未启动

**解决**:
```bash
cd admin
npm run dev
```

### 4. API路径404错误

**原因**: 环境配置文件错误

**解决**: 检查`admin/.env.development`文件是否存在且配置正确
